    数据保护与使用说明¶

    一、数据保护与使用¶

    1.1 信息收集范围¶

    我们仅收集为提供服务所必需的数据，具体包括：

    • 身份信息
    • 学校邮箱
    • 学号 / 工号

    • 在校状态

      用途仅限于身份验证与账号管理

    • 使用日志

    • 登录 IP
    • 时间戳
    • 操作类型（如文件上传 / 下载 / 删除）

    • 留存期限：180 天

    • 文件内容

    • 用户存储的文件默认加密存储
    • 仅对账户持有人可见

    管理员访问说明： - 仅在以下情形下，管理员可能依法访问文件内容： - 司法机关依法提出要求 - 校园安全事件调查与处置 - 平台不进行日常性或主动扫描私人文件内容

    1.2 数据用途¶

    数据仅用于以下合法、正当目的：

    • 核心服务
    • 文件存储

    • 跨设备同步等云盘基础功能

    • 安全保障

    • 异常登录检测
    • 多因素认证

    • 风险行为识别

    • 服务优化

    • 基于匿名化、去标识化数据进行统计分析
    • 用于功能改进与性能优化

    二、共享文件与安全机制¶

    2.1 内容安全检测¶

    • 所有通过公开链接分享的文件，将接入第三方安全引擎（如阿里云内容安全）进行检测
    • 若识别到潜在违法或违规内容（如暴力、违禁品信息等），系统将：
    • 自动拦截分享行为
    • 通过系统消息或邮件通知用户

    2.2 知识产权保护¶

    禁止上传的内容包括但不限于：

    • 盗版教材
    • 未经授权的软件、影视、音频资源
    • 其他违反《信息网络传播权保护条例》的内容

    侵权处理机制：

    • 收到权利人提交的有效投诉后，平台将：
    • 下架相关文件
    • 通知上传者处理结果
    • 保留处理记录，用于后续申诉或核查

    三、用户责任与权利¶

    3.1 合规使用要求¶

    用户在使用云盘服务过程中，应遵守国家法律法规及学校相关管理规定，禁止以下行为（包括但不限于）：

    • 存储或传播国家秘密
    • 存储或传播他人个人隐私数据
    • 利用云盘服务发起网络攻击
    • 传播恶意软件、病毒或攻击脚本

    3.2 隐私控制¶

    • 用户在分享文件时，应主动设置访问权限，例如：
    • 仅本校成员可查看
    • 指定人员可编辑
    • 分享链接默认有效期建议为 7 天
    • 系统支持用户自定义分享链接有效期限

    3.3 用户权利¶

    用户依法享有以下权利：

    • 数据导出

    • 可随时下载个人文件，用于本地备份

    • 账号注销

    • 毕业或离职后，可申请删除账户及其关联数据
